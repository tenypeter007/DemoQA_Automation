# DemoQA_Automation

A  framework based on Page Object Model, Selenium, TestNG using Java.

The framework uses:

- Java
- Selenium
- Rest Assured
- TestNG
- ExtentReport
- Log4j

## Execution 
```mvn clean test```

## Reporting
- Log - In file logfile.log.
- A html report - Which is generated using extent reports, under the folder ExtentReports.
